Hospital Management System

Project Overview

Hospital Management System is a web-based application developed to manage hospital activities efficiently. The system helps manage patients, doctors, appointments, billing, and medical records in one platform.

Features

Patient Registration

Doctor Management

Appointment Booking

Billing System

Medical Records Management

Admin Dashboard

Search and Filter Options


Technologies Used

Frontend: HTML, CSS, JavaScript, Bootstrap

Backend: Java / JSP / Servlet

Database: MySQL

IDE: Eclipse / IntelliJ IDEA

Server: Apache Tomcat


Modules

1. Admin Module

Manage doctors

Manage patients

View reports

Manage appointments


2. Doctor Module

View patient details

Manage prescriptions

Check appointments


3. Patient Module

Register and login

Book appointments

View prescriptions

Check billing details


Database

The project uses MySQL database to store:

Patient Details

Doctor Details

Appointment Information

Billing Records

Login Credentials


Installation Steps

1. Download or clone the repository.


2. Open the project in Eclipse or IntelliJ IDEA.


3. Import the database into MySQL.


4. Configure database connection settings.


5. Run the project using Apache Tomcat server.



How to Run

1. Start MySQL server.


2. Start Apache Tomcat.


3. Run the project.


4. Open browser and visit:



http://localhost:8080/HospitalManagementSystem
